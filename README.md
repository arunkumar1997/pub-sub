# Pub Sub app


## Installation

1. Install docker and docker compose https://docs.docker.com/engine/install/

2. Install node modules 
    ```sh
   yarn install
   ```

3. Build the app
    ```sh
    yarn build
    ```

4. start the database
    ```sh
    yarn start:db
    ```

5. start the app 
    ```sh
    yarn start:prod
    ```
